# 云服务商集成的starter包
> 阿里云\
> 腾讯云\
> 华为云\
> 七牛云\
> 其他 本地化或者私有云方案  包含minio

引入依赖
```
<dependency>
    <groupId>com.moensun.cloud.integration</groupId>
    <artifactId>cloud-integration-spring-boot-starters</artifactId>
    <version>1.1-SNAPSHOT</version>
    <type>pom</type>
    <scope>import</scope>
</dependency>
```
